# Insurance Data Analysis Dashboard | Power BI

## 📌 Project Overview

This project focuses on analyzing insurance data using **Microsoft Power BI** to create an interactive dashboard and generate meaningful business insights.

The dashboard helps analyze customer behavior, policy performance, revenue trends, and key business metrics for better decision-making.

---

## 🎯 Objectives

- Analyze insurance business performance
- Track important KPIs
- Understand customer and policy trends
- Create interactive visualizations
- Generate data-driven insights

---

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Visualization
- GitHub

---

## 📂 Dataset

The dataset contains insurance-related information such as:

- Customer details
- Policy information
- Premium values
- Revenue data
- Insurance transactions

---

## 🔄 Data Processing

Performed data preparation using Power Query:

- Removed duplicates
- Handled missing values
- Cleaned and transformed data
- Prepared data model for analysis

---

## 📊 Dashboard Features

### Insurance Overview Dashboard

- Total Revenue
- Total Customers
- Total Policies
- Average Premium
- Policy analysis
- Customer insights


### Detailed Analysis Dashboard

- Customer segmentation
- Policy performance
- Revenue trends
- Interactive filters and slicers

---

## 📈 Key DAX Measures

Created measures for:

- Total Revenue
- Total Customers
- Total Policies
- Average Premium Value
